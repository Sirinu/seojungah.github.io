## 💾 git 저장소 사용법

```bash
$ git clone https://github.com/seojungah/seojungah.github.io.git
```
### git pull
저장소에 변경된 코드를 가져온다.

```bash
$ git pull
```

### git commit
vscode의 git을 이용한다.

```bash
$ git commit [ -m COMMENT ]
```

### git push
```bash
$ git push
```
